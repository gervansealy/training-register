STEP 1: Configure Firebase
---------------------------
1. Copy firebase-config.template.js to firebase-config.js
2. Add your Firebase project credentials
   (Get them from Firebase Console > Project Settings)

STEP 2: Initialize Git & Push to GitHub
----------------------------------------
Run this PowerShell script:
   .\initialize-git.ps1

Or manually:
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/training-register.git
   git push -u origin main

STEP 3: Enable GitHub Pages
----------------------------
1. Go to your GitHub repository
2. Settings > Pages
3. Source: main branch, / (root)
4. Save

Your site will be live at:
https://YOUR_USERNAME.github.io/training-register/
